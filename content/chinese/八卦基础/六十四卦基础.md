## 八卦来历

大家都知道 `太极生两仪,两仪生四象,四象生八卦` 那么怎么来的呢,太极生出一阴一阳,
称为阴爻和阳爻,阴爻和阳爻copy一份,出来四个,在四个的上方分别放一个阴阳,得出四象.
同理可得得出八卦

- 太极：&#x262F;
- 阴爻：&#x268B;
- 阳爻：&#x268A;
- 太阳：&#x268C;
- 少阴：&#x268D; 
- 少阳：&#x268E; 
- 太阴：&#x268F; 



|  名称 | 含义               |
|  ---- | ------------------ |
|  太极 | 宇宙本源，阴阳未分 |
| 阴爻 | 阴仪，柔顺之气     |
|阳爻 | 阳仪，刚健之气     |
| 太阳 | 老阳，至阳         |
|  少阴 | 阳中生阴，阴渐长   |
| 少阳 | 阴中生阳，阳渐长   |
|太阴 | 老阴，至阴         |
